# 庫存管理系統

這是一個可直接啟動的簡易庫存管理系統，使用 Python 內建模組實作，不需要安裝第三方套件。

## 目前功能

- 登入頁，支援帳號與密碼驗證
- 30 分鐘閒置自動登出
- 預設管理員帳號初始化
- 使用者可修改自己的密碼
- 管理員可新增使用者並指定角色
- 角色權限區分：`admin`、`manager`、`operator`、`viewer`
- 商品建立
- 入庫與出庫
- 庫存異動紀錄
- 低庫存提醒

## 預設管理員帳號

- 帳號：`admin`
- 密碼：`Admin@123456`

## 啟動方式

```bash
python3 app.py
```

啟動後開啟：

[http://127.0.0.1:8000](http://127.0.0.1:8000)

如果要改 port：

```bash
PORT=8080 python3 app.py
```

## 系統資料

- SQLite 資料庫位置：`data/inventory.db`

如果你要把這套系統真正用在公司流程，下一步建議補這些：

1. 密碼修改與重設流程
